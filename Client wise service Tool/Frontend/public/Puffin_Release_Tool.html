🔧 Option A — Power Automate HTTP trigger (recommended):
1. Go to make.powerautomate.com → Create → Instant cloud flow
2. Trigger: "When an HTTP request is received" → Method = POST → Save to get the URL
3. Add action: Microsoft Teams → Post message in a chat or channel
4. In the Message field paste: @{triggerBody()?['text']}
5. Save → copy the HTTP POST URL from the trigger step → paste above

🔗 Option B — Teams Incoming Webhook (simpler, no Power Automate needed):
1. In Teams, go to the target channel/group → click ··· → Connectors → Incoming Webhook
2. Configure → copy the webhook URL (starts with https://xxx.webhook.office.com) → paste above
⚠ Note: Incoming Webhooks only work with Teams channels, not group chats. Use Power Automate for group chats.
